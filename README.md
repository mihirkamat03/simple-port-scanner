# Simple Port Scanner

A lightweight, Python-based TCP port scanner designed for network security. This tool helps identify open ports on target IP addresses, supporting both single and multiple target scanning.

## Features
- **TCP Connect Scan:** Checks for open ports using standard socket connections.
- **Multiple Targets:** capable of scanning multiple IP addresses simultaneously (comma-separated).
- **User Friendly:** Simple command-line interface for easy interaction.
